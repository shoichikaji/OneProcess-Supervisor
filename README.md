# NAME

OneProcess::Supervisor - supervisor for 1 process

# SYNOPSIS

    > supervisor.pl --daemonize --pid foo.pid --log foo.log ruby long-time-script.rb

# DESCRIPTION

OneProcess::Supervisor is a simple supervisor for 1 process management.
